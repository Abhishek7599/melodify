# 🎧 Melodify

**Melodify** is a sleek and modern web-based music player that allows users to search and play 30-second song previews using the iTunes API. It offers a smooth and interactive experience with essential playback controls and smart song suggestions.

---

## 🚀 Live Demo
https://abhishek7599.github.io/melodify/
---

## ✨ Features

* 🔍 **Search Songs**
  Instantly search for your favorite tracks using the iTunes API

* ▶️ **30-Second Preview Playback**
  Listen to song previews directly in the browser

* ⏯️ **Play / Pause Toggle**
  Seamless control over playback

* ⏭️ **Next / Previous Buttons**
  Easily navigate between songs

* 💡 **Suggested Songs**
  Get related or recommended tracks

* 📱 **Responsive Design**
  Works smoothly on desktop and mobile devices

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive design
* **JavaScript (Vanilla JS)** – Functionality and API handling
* **iTunes Search API** – Fetching songs and preview URLs

---

## 📦 How It Works

1. User enters a song name in the search bar
2. App sends a request to the iTunes API
3. Songs are displayed dynamically
4. Clicking a song plays a 30-second preview
5. Users can navigate using next/previous controls
